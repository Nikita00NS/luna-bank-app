# 🌙 Luna Bank v1.2

**Crypto Financial Ecosystem** — Telegram Mini App

## Features
- 32 screens, glass-morphism design
- Supabase backend (9 tables)
- TON Connect wallet integration
- Electronic signature + PDF contracts
- Luna City game, NFT marketplace, Mini-games
- Escrow service, Swap, Pro Exchange, Staking
- Telegram bot notifications (@LunaBankBot)

## Tech Stack
- React 19 + TypeScript + Vite + Tailwind CSS
- Supabase (PostgreSQL + Storage)
- jsPDF for document generation
- Zustand for state management

## Setup
```bash
npm install
npm run dev
```

## Deploy
```bash
npm run build
# Deploy dist/ to Vercel
```
